# Multiplayer Demo
A starting point for a 3D multiplayer game written in JavaScript.
### What's in this repository?
- `client3d`: The NW.js- and three.js-powered client for the multiplayer game
- `client.js`: The old 2D client, runs on node.js and uses the [xgfx](https://github.com/thecoder08/xgfx) library for graphics
- `server.js`: The multiplayer server, runs on node.js
